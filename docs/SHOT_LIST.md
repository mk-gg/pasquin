# Screenshot shot list

The README references three images in this folder. Capture them from the live
site (https://pasquin.mkgg.dev) or a local `npm run dev`, save them here with
the **exact filenames** below, then commit and push.

Tips: use a desktop-width window (~1280px), the dark theme looks sharpest, and
crop to the app area (you don't need the whole browser chrome).

| Filename | What to show |
| --- | --- |
| `editor.png` | The home page with the editor and toolbar. Type a few lines with some formatting (a heading, a bullet list, some bold) so it looks alive. |
| `share.png` | An owned note's **Share** dialog with "Allow editing" on, showing the link and the **QR code**. (Save a note, then click Share.) |
| `viewer.png` | A shared note open in **read-only** view — the title, the expiry badge, and the content. |

Optional extras you could add later: the "My Notes" sidebar, the save dialog
with password protection, the mobile layout.
